免责声明
--------------------
本开发工具包中的代码只是为了方便商户测试而提供的示例代码，商户可根
据自己的需要，按照技术规范编码实现，并非一定要使用该工具包中的代码。

该代码仅供参考。


使用说明
--------------------
    1. 首先，请仔细阅读《UPMP商户接入技术改造指南》
    2. 根据您的开发语言选择相应版本的工具包（含示例代码）
    3. 开始编码
    4. 编码过程中，如有疑问，请在QQ支持群提出
    5. 技术人员私聊支持
